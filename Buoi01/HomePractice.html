HTML đã tương đối nên trong lần practice này chúng ta tập trung vào luyện tập CSS. Chưa luyện tập Javascript vì chưa học. Haha.

Về selector trong CSS chúng ta tập trung vào 03 loại chính:

  1. Type selector:
    • Định nghĩa: định dạng khớp với tên element
    • Cú pháp: tên_tag, tên_tag {nội_dung}
  2. Class selector
    • Định nghĩa: định dạng element có có attribute khớp với attribute được đề cập sau dấu chấm
    • Cú pháp:
      • .tên_class {nội_dung}: định dạng các element có class là tên_class
      • tên_element.tên_class {nội_dung}: định dạng các "tên_element" có class là "tên_class"

        • ==> Từ trên suy ra, mình nghĩ nó có thể chấm nhiều lần để định dạng vào sâu hơn. Và nếu như nhớ không lầm thì ta có thể kết hợp các kiểu selector này với nhau.

  3. ID selector
    • Định nghĩa: định dạng các element có giá trị khớp với giá trị được đề cập sau dấu #
    • Cú pháp: #tên_ID {nội_dung}
